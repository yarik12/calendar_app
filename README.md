# Advanced Calendar Application

Этот проект — простое десктопное приложение для планирования дел, написанное на
Python с использованием библиотеки **Flet**. Приложение позволяет вести
каледарь задач и заметок на каждый день, отмечать их выполнение, переносить на
другие даты, а также создавать регулярно повторяющиеся мероприятия.

## Возможности

- **Просмотр задач по дням**: выбрать дату можно с помощью кнопки с пиктограммой
  календаря. Список задач для выбранного дня отображается в виде карточек.
- **Добавление задач**: доступны поля для ввода названия, описания и времени
  выполнения. Можно задать периодичность: однократно, ежедневно, еженедельно
  или ежемесячно.
- **Перенос задач**: однократную задачу можно перенести на другую дату через
  кнопку со стрелкой. Выбирается новая дата, и задача перемещается.
- **Отметка выполнения**: однократные задачи отмечаются чекбоксом.
- **Сохранение данных**: все задачи и мероприятия сохраняются в файл
  `calendar_data.json` рядом с исполняемым скриптом и восстанавливаются при
  следующем запуске.

## Запуск

```bash
pip install -r requirements.txt
python calendar_app.py
```

При запуске с помощью `flet` можно открывать приложение в окне системы:

```bash
flet run calendar_app.py
```

## Структура кода

- **Task** — класс, представляющий одно задание или заметку. Содержит
  информацию о названии, описании, дате, времени, состоянии выполнения и
  уникальном идентификаторе.
- **RecurringTask** — наследник `Task`, дополняющий его периодичностью
  (ежедневно, еженедельно, ежемесячно) и датой начала. Метод `occurs_on`
  определяет, нужно ли отображать задачу в конкретный день.
- **CalendarManager** — центральный класс, отвечающий за хранение, загрузку и
  сохранение данных. Управляет списком однократных задач и регулярных
  мероприятий, предоставляет методы для добавления, удаления, переноса и
  отметки задач.
- **CalendarApp** — графический интерфейс, реализованный на Flet. Подписывает
  обработчики на события и связывает их с функциями `CalendarManager`.

## Зависимости

Требуется только библиотека `flet`, которая устанавливается из PyPI.
